# Appendix

## CoC ID Format

Canonical CoC IDs use:

- `{STATE}-{NNN}`
- Example: `CO-500`, `NY-600`

Cross-state PIT variants like `MO-604a` are normalized to canonical CoC IDs during PIT parsing.

## Coordinate System

Boundary and census geometry artifacts are stored in EPSG:4326.

## Geometry Hashes

Boundary normalization computes geometry hashes for change detection across vintages.

## Temporal Notation

See [[08-Temporal-Terminology|Temporal Terminology]] for shorthand (`A`, `B`, `C`, `T`, `P`, `Y`) and compound notation conventions.
