# CoC Lab

CoC Lab is a Python-based data and geospatial infrastructure for working with Continuum of Care (CoC) boundary data. It provides tools to ingest, validate, version, and visualize CoC boundaries from HUD data sources.  It includes the ability to aggregate American Community Survey (ACS) information, along with Zillow Observed Rent Index (ZORI), to those CoC boundaries.

Refer to the manual in the /manual-obsidian directory for instructions on installation and operation.
